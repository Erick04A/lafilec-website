## About the Project

LA FIL ec needed a complete digital presence to communicate their identity, promote their events and connect with their musical community. This platform delivers exactly that — built fast, designed clean and deployed to production.

This was a real freelance engagement handled end-to-end: from the initial brief and architecture decisions through to deployment and ongoing maintenance. No team, no templates — built from the ground up.

## Running Locally

```bash
# Clone this repository
git clone https://github.com/Erick04A/lafilec-website.git

# Move into the project
cd lafilec-website

# Install dependencies
npm install

# Start the development server
npm run dev

# Open in browser at http://localhost:5173
```
